== 專案使用手冊

請您使用 IDE 的 Tomcat 或是外部的 Tomcat 開啟。

=== tomcat (Intellij)

點選右上角編譯旁邊的edit configuration，新增tomcat並選擇版本10.1.23，選擇完版本後進入此tomcat設定中的deployment，點擊 "+"這個符號後選擇Artifact
之後會有一個選單，請選擇"hwajax:war"，並且把下方的Application context改成 "/hwajax"
